As mentioned in the description, this is a port of the repository at https://github.com/cinquemb/WifiBruteCrack. I made minimal modifications, primarily adding a redundant password list. The tool primarily generates credentials by pairing lines from nouns.txt and adjectives.txt in the format "adjective + noun," followed by three digits, which is the standard pattern for ISPs in my area. It automatically selects all visible SSIDs, starting with the strongest signal, generates a list of 5,000 passwords, and tests five different capitalization variants for each.
